# Model Details

The following table summarizes the core technical specifications and foundational properties of the GLM-5.2 model as officially published on the Hugging Face model card.

| Property | Value |
| :--- | :--- |
| **Developer** | Z.ai |
| **Model Type** | Text Generation |
| **Parameters** | 753 Billion (Total) |
| **Context Length** | 1 Million Tokens |
| **Languages** | English, Chinese |
| **License** | MIT |

---

### Key Takeaways
* **Massive Scale:** With a total footprint of 753 Billion parameters, this model leverages scale to handle highly intricate logical deductions and complex multi-turn reasoning paths.
* **Ultra-Long Context:** The native 1-million-token context window allows the model to process massive codebases, multi-hour meeting transcripts, or entire legal portfolios in a single prompt.
* **Permissive Open-Source Licensing:** Falling under the **MIT License**, developers and organizations have the absolute freedom to use, modify, distribute, and commercialize the model without hitting restrictive legal boundaries.
